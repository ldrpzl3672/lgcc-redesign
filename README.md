# LGCC Redesign (Proposal)

This is a lightweight, static site proposal for Lebanon Golf & Country Club.

## Quick start (local)
Open `index.html` in a browser, or run a tiny server:

```bash
python -m http.server 8000
```

Then visit http://localhost:8000

## Deploy options
### GitHub Pages
- Create a repo
- Upload these files to the repo root
- Settings → Pages → Deploy from branch → `main` / root

### Netlify
- Drag & drop the folder in Netlify “Deploy manually”
- Or connect GitHub repo

### Vercel
- Import the repo as a static site

## Before launch
Replace placeholder values in `index.html`:
- Phone
- Address
- Facebook URL
- Menu link / hours
Replace images in `/assets`:
- hero.jpg
- course-1.jpg
- dining-1.jpg
- og-image.jpg
- favicon.png
